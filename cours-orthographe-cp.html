✏️ Orthographe

Majuscule · Pluriel · Féminin · Sons

📌 La majuscule
On met une majuscule : au début d'une phrase · pour les prénoms · pour les villes.
Ex : Léa habite à Lyon.
📌 Le pluriel
On ajoute -s au nom (en général). Les mots en -eau font -eaux au pluriel.
Ex : un chat → des chats · un oiseau → des oiseaux
📌 Le féminin
On ajoute -e (en général). Mais certains mots changent davantage.
Ex : un ami → une amie · un chanteur → une chanteuse · le boulanger → la boulangère
📌 Le son [s]
s (début de mot) · ss (entre 2 voyelles) · c (devant e, i) · ç (devant a, o, u).
Ex : sauter · poisson · cerise · garçon

📌 À retenir

  • Majuscule : début de phrase, prénom, ville
  • Pluriel : + -s (oiseau → oiseaux)
  • Féminin : + -e en général
  • Son [s] : s / ss / c / ç selon la lettre suivante
