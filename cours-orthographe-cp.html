✏️ Orthographe

Majuscule · Pluriel · Féminin · Sons

1. La majuscule

Majuscule = 1re lettre d'une phrase · Prénoms · Noms de villes et pays
Exemples
  • Début de phrase : Le chat dort.
  • Prénom : Léa · Théo · Emma
  • Ville / pays : Paris · France · Lyon

2. Le pluriel

Règle générale : + -s

Un chat → des chats · une maison → des maisons

Exceptions à connaître
SingulierPlurielRègle
un oiseaudes oiseauxnoms en -eau → -eaux
un jeudes jeuxnoms en -eu → -eux
une sourisdes sourisdéjà en -s → invariable
un nezdes nezdéjà en -z → invariable

3. Le féminin

Règle générale : + -e

un ami → une amie · un petit → une petite

Féminins particuliers
MasculinFémininRègle
un chanteurune chanteuse-eur → -euse
un boulangerune boulangère-er → -ère
un acteurune actrice-eur → -rice (certains)
un chatune chatteconsonne double
un lionune lionneconsonne double
un princeune princesse-ce → -cesse

4. Le son [s]

s (début) · ss (entre 2 voyelles) · c (devant e, i, y) · ç (devant a, o, u)
LettreQuand ?Exemples
sDébut de mot, après consonnesauter · sable · danser
ssEntre deux voyellespoisson · tasse · dessert
cDevant e, i, ycerise · cirque · cycle
çDevant a, o, ugarçon · reçu · façon
Attention !

s entre deux voyelles → son [z] : maison, cuisine, rose
Pour le son [s] entre deux voyelles → il faut écrire ss : poisson, tasse

📌 À retenir

  • Majuscule : début de phrase, prénoms, noms de villes/pays
  • Pluriel : + -s en général ; -eaux pour les mots en -eau
  • Féminin : + -e en général ; -eur → -euse, -er → -ère
  • Son [s] : ss entre 2 voyelles · c devant e/i · ç devant a/o/u
  • s entre 2 voyelles → son [z] (maison = [z])
