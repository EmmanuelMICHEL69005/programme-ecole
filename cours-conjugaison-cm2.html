1. Le passé simple

Usage

Le passé simple s'utilise principalement à l'écrit dans les récits littéraires. Il exprime une action brève et accomplie dans le passé (s'oppose à l'imparfait qui décrit).

2. Le plus-que-parfait

Formation et usage

Auxiliaire avoir ou être à l'imparfait + participe passé
Exprime une action antérieure à une autre action passée.

  • Quand il arriva, elle était déjà partie. (le départ = avant l'arrivée)
  • Il avait mangé avant de sortir.

3. Le conditionnel présent

Formation

Base du futur + terminaisons de l'imparfait : -ais/-ais/-ait/-ions/-iez/-aient

Usages du conditionnel
  • Hypothèse : Si j'avais le temps, je lirais davantage.
  • Politesse : Pourrais-tu m'aider ?
  • Information non vérifiée : Il serait malade. (selon certaines sources)

4. La concordance des temps (notion)

Quand utiliser quel temps ?
Action principaleAction antérieureAction postérieure
Passé composé / Passé simplePlus-que-parfaitConditionnel
Ex. : Il arriva (PS) — il avait préparé tout (PQP) — il pensait qu'il trouverait (COND) facilement.

📌 À retenir

  • Passé simple : action brève dans un récit → 1er gr. : -ai/-as/-a/-âmes/-âtes/-èrent
  • Passé simple de être : fus/fus/fut/fûmes/fûtes/furent
  • Plus-que-parfait : imparfait d'avoir/être + PP → action avant une autre action passée
  • Conditionnel : base futur + terminaisons imparfait → hypothèse ou politesse
