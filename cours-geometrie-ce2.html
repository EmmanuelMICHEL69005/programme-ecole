🔷 Géométrie

Figures planes, périmètre, symétrie, coordonnées

📌 Figures planes
Carré : 4 côtés égaux + 4 angles droits. Rectangle : côtés opposés égaux + 4 angles droits. Triangle : 3 côtés. Cercle : centre + rayon.
Ex : carré de 5 cm → P = 4×5 = 20 cm · rectangle 5×3 → P = 2×(5+3) = 16 cm
📌 Périmètre
Le périmètre = somme de tous les côtés. Carré : P = 4×c · Rectangle : P = 2×(L+l) · Triangle : P = a+b+c.
Ex : triangle 3+4+5 = 12 cm · rectangle 6×4 → P = 2×(6+4) = 20 cm
📌 Axe de symétrie
Une figure est symétrique si on peut la plier pour que les deux parties se superposent exactement. Carré : 4 axes · Rectangle : 2 axes · Cercle : infini.
Ex : carré a 4 axes · rectangle n'a pas d'axe diagonal
📌 Coordonnées
Sur un quadrillage, on repère un point par (colonne, ligne). La colonne s'écrit en premier.
Ex : point A en (3, 2) → 3e colonne, 2e ligne

📌 À retenir

  • Périmètre = somme des côtés · carré : 4×c · rectangle : 2×(L+l)
  • Axe de symétrie = ligne de pliage
  • Coordonnées (x, y) : colonne puis ligne
