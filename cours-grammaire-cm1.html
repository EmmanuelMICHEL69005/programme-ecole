Fonctions, types de phrases, nature et fonction

1. Types et formes de phrases

Les 3 types de phrases
TypeUsagePonctuationExemple
DéclarativeDonne une information.Il pleut aujourd'hui.
InterrogativePose une question?Vient-il avec nous ?
ImpérativeDonne un ordre, conseil. ou !Ferme la porte !
Les 2 formes de phrases
  • Affirmative : Elle mange une pomme.
  • Négative : Elle ne mange pas de pomme. (ne…pas, ne…plus, ne…jamais, ne…rien)
Forme exclamative

S'ajoute à n'importe quel type : Quel beau temps ! — Comme il chante bien !

2. Les fonctions dans la phrase

Le groupe sujet (GS)

Répond à Qui est-ce qui… ? ou Qu'est-ce qui… ? Il commande le verbe.

  • Les enfants jouent dans le jardin. → Qui joue ? Les enfants.
Le complément d'objet direct (COD)

Répond à quoi ? ou qui ? directement après le verbe. Pas de préposition.

PhraseQuestionCOD
Léo lit un roman.Léo lit quoi ?un roman
Elle regarde sa sœur.Elle regarde qui ?sa sœur
Le complément d'objet indirect (COI)

Répond à à qui ?, de quoi ?, à quoi ? — avec une préposition (à, de).

PhraseQuestionCOI
Il parle à sa mère.Il parle à qui ?à sa mère
Elle rêve de vacances.Elle rêve de quoi ?de vacances

3. Les compléments circonstanciels (CC)

Caractéristiques

Les CC donnent des circonstances de l'action. Ils sont :

  • Déplaçables : Dans le jardin, les enfants jouent.
  • Supprimables : Les enfants jouent. (phrase encore correcte)
TypeQuestionExemples
CC de lieuOù ?dans le jardin, à Paris, sur la table
CC de tempsQuand ?demain, le soir, pendant l'été
CC de manièreComment ?rapidement, avec soin, en courant

4. Nature et fonction

Distinction essentielle
  • La nature (classe grammaticale) = ce qu'est le mot → nom, verbe, adjectif, adverbe, préposition…
  • La fonction = le rôle du mot dans la phrase → sujet, COD, COI, CC, épithète…

Un même mot peut avoir des fonctions différentes selon la phrase :

  • Le chat dort. → "chat" = nom commun (nature), sujet (fonction)
  • Il caresse le chat. → "chat" = nom commun (nature), COD (fonction)

5. Le groupe nominal étendu

Structures du GN
  • Dét. + Nom : le chien
  • Dét. + Adj. + Nom : le gros chien
  • Dét. + Nom + Adj. : le chien noir (adjectif épithète)
  • Dét. + Nom + complément du nom : le chien de la voisine

📌 À retenir

  • 3 types de phrases : déclarative · interrogative · impérative
  • 2 formes : affirmative / négative
  • COD : quoi/qui directement → sans préposition
  • COI : à qui ?/de quoi ? → avec préposition (à, de)
  • CC : lieu, temps, manière → déplaçable et supprimable
  • Nature = ce qu'est le mot ; Fonction = son rôle dans la phrase
