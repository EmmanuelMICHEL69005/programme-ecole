📐 Les angles

Types d'angles, rapporteur, angles dans les figures

📌 Types d'angles
Droit (90°) · aigu (< 90°) · obtus (> 90°) · plat (180°).
Ex : coin d'une feuille = 90° (droit) · angle de 35° = aigu · angle de 120° = obtus
📌 Mesurer avec le rapporteur
Placer le centre du rapporteur sur le sommet de l'angle, aligner un côté sur 0°, lire la graduation.
Ex : angle de 35° → aigu · angle de 110° → obtus
📌 Angles dans les figures
Dans un triangle, la somme des 3 angles vaut toujours 180°.
Ex : triangle équilatéral : 3 × 60° = 180° · triangle rectangle : 90° + 50° + 40° = 180°

📌 À retenir

  • Droit 90° / aigu <90° / obtus >90° / plat 180°
  • Rapporteur : centre sur le sommet
  • Triangle : somme des angles = 180°
