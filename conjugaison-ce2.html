📝 CE2 — Conjugaison
💡 Je retiens
📝 1er groupe : verbes en -ER (ex : manger, jouer) → je joue, tu joues…
📗 2e groupe : verbes en -IR (ex : finir, grandir) → je finis, tu finis…
🕐 Présent : action qui se passe maintenant
🕑 Imparfait : action passée qui durait (je jouais…)
🕒 Passé composé : avoir/être + participe passé (j'ai joué / je suis allé)
🕓 Futur simple : action à venir, terminaisons -rai, -ras, -ra…
