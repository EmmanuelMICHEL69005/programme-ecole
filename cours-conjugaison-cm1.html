3. Imparfait de l'indicatif

Terminaisons (tous groupes)

Radical de la 1re pers. du pluriel (présent) + -ais / -ais / -ait / -ions / -iez / -aient

Marque de temps de l'imparfait

La marque de temps de l'imparfait est -ai- (je chantais) ou -i- pour les verbes en -ier (nous étudiions).

4. Futur simple

Terminaisons (tous groupes)

Base du futur (= infinitif en général) + -ai / -as / -a / -ons / -ez / -ont
La marque de temps du futur est -r-.

Bases irrégulières au futur

voir → verr- · venir → viendr- · pouvoir → pourr- · vouloir → voudr- · prendre → prendr-

5. Passé composé

Formation

Auxiliaire avoir ou être (au présent) + participe passé

Avec avoir (la majorité des verbes)

J'ai chanté · tu as fini · il a fait · nous avons pris · vous avez vu · ils ont voulu

Avec être (déplacement + pronominaux)

aller/venir · partir/arriver · entrer/sortir · monter/descendre · naître/mourir · tomber · rester
→ Le participe passé s'accorde avec le sujet : Elle est partie. Ils sont venus.

📌 À retenir

  • Imparfait : marque -ai- → je chantais, il finissait, nous faisions
  • Futur : marque -r- → je chanterai, il ira, nous ferons
  • Passé composé : avoir/être + PP — avec être : PP s'accorde avec le sujet
  • Irréguliers : faire/fait, voir/vu, prendre/pris, vouloir/voulu, pouvoir/pu
  • Futur irrégulier : voir→verr-, venir→viendr-, pouvoir→pourr-, aller→ir-
