CM1

📝 Conjugaison

Présent, imparfait, futur, passé composé

📌 Présent (révision)
Irréguliers à retenir par cœur : faire / aller / venir / voir / pouvoir / vouloir / prendre.
Ex : je veux · tu peux · il vient · nous allons · vous faites · ils prennent
📌 Imparfait
Radical du présent + terminaisons : -ais / -ais / -ait / -ions / -iez / -aient. Marque : -ai- devant la terminaison.
Ex : je chantais · il finissait · nous voyions · vous parliez
📌 Futur simple
Infinitif + terminaisons : -rai / -ras / -ra / -rons / -rez / -ront. Bases irrégulières : être→ser- · avoir→aur- · aller→ir-.
Ex : il sera · j'aurai · nous irons · vous ferez · ils viendront
📌 Passé composé
Avoir ou être + participe passé. Avec être : le participe s'accorde avec le sujet.
Ex : Elle est arrivée. · Ils sont partis. · Nous avons fini. · J'ai vu.

📌 À retenir

  • Présent : irréguliers par cœur (vouloir, pouvoir, venir…)
  • Imparfait : -ais / -ait / -ions / -aient
  • Futur : -rai / -rons / -ront · ser-, aur-, ir- (irréguliers)
  • Passé composé : avoir ou être + PP · accord avec être
