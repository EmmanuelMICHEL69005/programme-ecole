🔤 Grammaire

Nature des mots, GN, types de phrases, accords

1. Les classes grammaticales (nature des mots)

🔷 Nom
personne, animal, chose
chat, Paris, maison
🟢 Verbe
action ou état
manger, être, courir
🟡 Adjectif
précise le nom
grand, rouge, joli
🔴 Déterminant
accompagne le nom
le, la, un, mon, ce
🟠 Pronom
remplace un nom
je, il, nous, y, en
🟣 Adverbe
modifie verbe/adj
vite, très, souvent
Nom commun vs nom propre
  • Commun : désigne une catégorie → ville, enfant, chien
  • Propre : désigne un individu précis, prend une majuscule → Paris, Lucas, France

2. Le groupe nominal (GN)

Structure du GN

Déterminant + Nom (+ adjectif, + complément du nom)

Exemples de GN
  • le chat → déterminant + nom
  • un grand chat → déterminant + adjectif + nom (ou déterminant + nom + adjectif)
  • les maisons de la ville → déterminant + nom + complément du nom
Accord dans le GN

Le déterminant et l'adjectif s'accordent en genre (masculin/féminin) et en nombre (singulier/pluriel) avec le nom.

  • un petit chien → des petits chiens
  • une belle fleur → de belles fleurs

3. Les types de phrases

💬
Déclarative
Donne une information. Se termine par un point.
Il fait beau.
❓
Interrogative
Pose une question. Se termine par ?
Tu viens ?
❗
Exclamative
Exprime un sentiment fort. Se termine par !
Quel beau temps !
👉
Impérative
Donne un ordre ou un conseil. Pas de sujet.
Viens ici !

4. L'accord sujet-verbe

Règle

Le verbe s'accorde toujours avec son sujet (même personne, même nombre).

Exemples
  • Le chien court (3e pers. singulier)
  • Les chiens courent (3e pers. pluriel)
  • Lila et Tom jouent → ils = 3e pers. pluriel
  • Ma sœur et moi chantons → nous = 1re pers. pluriel
Piège

Quand le sujet est éloigné du verbe ou inversé, on peut se tromper.
Les enfants de la classe chantent (le sujet, c'est "les enfants", pas "la classe")

5. Phrases affirmatives et négatives

La négation : ne … pas
  • Affirmative : Je mange des bonbons.
  • Négative : Je ne mange pas de bonbons.
  • Attention à l'article : des → de (pas de bonbons)
Autres négations
  • ne … plus : Je ne mange plus de bonbons.
  • ne … jamais : Je ne mange jamais de bonbons.
  • ne … rien : Je ne mange rien.

6. Les fonctions dans la phrase

Le groupe sujet (GS)

Répond à la question Qui est-ce qui… ? ou Qu'est-ce qui… ?
C'est lui qui commande le verbe.

  • Les enfants jouent dans le jardin. → Qui joue ? Les enfants = GS
  • Le vent souffle fort. → Qu'est-ce qui souffle ? Le vent = GS
Le complément d'objet direct (COD)

Répond à qui ? ou quoi ? directement après le verbe (sans préposition).

  • Je lis un livre. → Je lis quoi ? un livre = COD
  • Il voit sa mère. → Il voit qui ? sa mère = COD
Les compléments circonstanciels (CC)

Donnent des précisions sur les circonstances de l'action. On peut souvent les déplacer ou les supprimer.

TypeQuestionExemple
CC de lieuOù ?Il joue dans le jardin.
CC de tempsQuand ?Il joue le soir.
CC de manièreComment ?Il joue avec joie.

📌 À retenir

  • 6 natures : nom, verbe, adjectif, déterminant, pronom, adverbe
  • GN : déterminant + nom (+ adjectif) → accord en genre et nombre
  • 4 types de phrases : déclarative, interrogative, exclamative, impérative
  • Accord sujet-verbe : toujours chercher le vrai sujet
  • GS = qui fait l'action (qui ? / qu'est-ce qui ?)
  • COD = quoi/qui directement après le verbe ; CC = où/quand/comment
