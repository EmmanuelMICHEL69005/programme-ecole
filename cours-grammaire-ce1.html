📖 Grammaire

La phrase · le groupe nominal · les accords

📝 Types de phrases
Il existe 3 types de phrases principaux.
TypeRôlePonctuationExemple
DéclaratifDonner une info.Il fait beau.
InterrogatifPoser une question?Fait-il beau ?
ExclamatifExprimer un sentiment!Quel beau temps !
💡 L’impératif donne un ordre : Viens !
🎯 Le sujet et le verbe
Le sujet fait l’action. Le verbe s’accorde avec le sujet.
Le petit chat [sujet] dort [verbe] sur le canapé.
Les enfants [sujet] jouent [verbe] dans la cour.
💡 Pour trouver le verbe : essaie de changer le temps — il mange → il mangeait.
🔤 Le groupe nominal (GN)
Un groupe nominal = déterminant + nom (+ parfois un adjectif).
le [dét.] + chien [nom] → le chien
une [dét.] + belle [adj.] + maison [nom] → une belle maison
DéterminantNomAdjectif
✅ L’accord en genre et en nombre
L’adjectif s’accorde en genre et en nombre avec le nom.
MasculinFéminin
Singulierun chien noirune chatte noire
Plurieldes chiens noirsdes chattes noires
💡 Féminin = +e · Pluriel = +s · Féminin pluriel = +es

📌 À retenir

  • Phrase : Majuscule → point / ? / !
  • Sujet : Qui est-ce qui ? · Verbe : l’action
  • GN = déterminant + nom (+ adjectif)
  • L’adjectif s’accorde avec le nom : genre + nombre
