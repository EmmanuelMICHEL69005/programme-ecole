🔤 Grammaire

La phrase, nature des mots, sujet, verbe

1. La phrase

Reconnaître une phrase

Une phrase est un groupe de mots qui a un sens complet. Elle :

  • commence par une majuscule
  • se termine par un point (. ? ! …)
  • contient au moins un verbe conjugué
Phrase ou pas phrase ?
  • ✓ Le chat mange sa pâtée. → phrase complète
  • ✗ le chat mange → pas de majuscule ni de point
  • ✗ Dans le jardin. → pas de verbe conjugué

2. Les classes grammaticales (nature des mots)

🔷 Nom
personne, animal, chose
chat, maison, Lucas
🟢 Verbe
action ou état
manger, être, sauter
🟡 Adjectif
précise le nom
grand, rouge, joli
🔴 Déterminant
accompagne le nom
le, la, un, mon, ce
🟠 Pronom
remplace un nom
je, il, nous, elle
🟣 Adverbe
modifie verbe/adj
vite, très, souvent
Nom commun et nom propre
  • Commun : désigne une catégorie → ville, chat, ami
  • Propre : désigne quelqu'un ou quelque chose de précis → Paris, Lucas, France — prend toujours une majuscule

3. Le verbe

Repérer le verbe

Le verbe exprime une action ou un état. C'est le mot qui change quand on change le temps de la phrase.

  • Le chien court. → Le chien courait. ← le verbe change
Trouver le verbe

Entoure le mot qui répond à "Qu'est-ce que fait … ?"

  • Le chat mange sa pâtée. → Le chat fait quoi ? → mange
  • Elle est contente. → Elle fait quoi / est dans quel état ? → est

4. Le sujet

Trouver le sujet

Le sujet répond à la question Qui est-ce qui… ? ou Qu'est-ce qui… ? posée avant le verbe.

  • Le chien court. → Qui court ? Le chien = sujet
  • Ma sœur chante. → Qui chante ? Ma sœur = sujet
Le sujet commande le verbe

Le verbe s'accorde toujours avec son sujet.

  • Sujet singulier → verbe singulier : Le chat mange.
  • Sujet pluriel → verbe pluriel : Les chats mangent.

5. Les types de phrases

💬
Déclarative
Donne une information. Se termine par un point.
Il fait beau.
❓
Interrogative
Pose une question. Se termine par ?
Tu viens ?
❗
Exclamative
Exprime un sentiment fort. Se termine par !
Quel beau temps !
👉
Impérative
Donne un ordre ou conseil. Sans pronom sujet.
Viens ici !

📌 À retenir

  • Phrase : majuscule + verbe conjugué + point
  • 4 natures principales : nom, verbe, adjectif, déterminant
  • Verbe : exprime action ou état, change avec le temps
  • Sujet : "qui ?" fait l'action — commande le verbe
  • 4 types de phrases : déclarative (.) interrogative (?) exclamative (!) impérative
