📐 Géométrie

Pythagore, triangles, angles, cercle, repérage

📌 Triangles et angles
Somme des angles d'un triangle = 180°. Théorème de Pythagore : dans un triangle rectangle, a² + b² = c² (c = hypoténuse, côté le plus long).
Ex : côtés 3, 4, 5 → 9 + 16 = 25 = 5² ✓ triangle rectangle
📌 Cercle
Circonférence = 2πr · Aire = πr² (π ≈ 3,14). r = rayon, d = diamètre = 2r.
Ex : r = 5 cm → C ≈ 31,4 cm · A ≈ 78,5 cm²
📌 Repérage dans le plan
Plan avec axes x (abscisse) et y (ordonnée). Point A(x ; y). Symétrie axiale : par rapport à un axe. Symétrie centrale : par rapport à un point.
Ex : A(2 ; 3) symétrique de A'(−2 ; 3) par l'axe des ordonnées

📌 À retenir

  • Angles d'un triangle = 180°
  • Pythagore : a² + b² = c² (triangle rectangle)
  • Cercle : circonférence = 2πr · aire = πr²
  • Repérage : point A(x ; y) dans le plan
