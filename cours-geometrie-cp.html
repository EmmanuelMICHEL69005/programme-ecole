📐 La géométrie

Formes, solides, espace et symétrie

📌 Les formes planes
Carré (4 côtés égaux) · Rectangle (2 paires de côtés égaux) · Triangle (3 côtés) · Cercle (rond).
Ex : une fenêtre = rectangle · une pizza = cercle
📌 Les solides
Cube (6 faces carrées) · Sphère (boule) · Cylindre (rouleau). Les solides ont du volume, on peut les tenir dans la main.
Ex : un dé = cube · un ballon = sphère
📌 Les lignes
Ligne droite · Ligne courbe · Ligne brisée. Une règle trace des lignes droites.
Ex : un bord de livre = ligne droite · un arc-en-ciel = courbe

📌 À retenir

  • Carré / rectangle / triangle / cercle (formes plates)
  • Cube / sphère / cylindre (solides en 3D)
  • Ligne droite, courbe, brisée
