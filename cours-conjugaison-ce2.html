📝 Conjugaison

Groupes, présent, imparfait, futur, passé composé

1. Les groupes de verbes

3 groupes
  • 1er groupe : infinitif en -ER (sauf aller) → manger, jouer, chanter
  • 2e groupe : infinitif en -IR, participe présent en -issant → finir (finissant), grandir, choisir
  • 3e groupe : tous les autres → aller, faire, être, avoir, vouloir, venir…
Attention !
  • Partir, courir, dormir → 3e groupe (même s'ils finissent en -IR)
  • Aller → 3e groupe (exception pour -ER)

Quand utiliser l'imparfait ?
  • Action qui durait dans le passé : Il faisait beau.
  • Habitude passée : Chaque matin, je mangeais des céréales.
  • Description dans le passé : La maison était grande.

5. Passé composé

Formule

Auxiliaire (être ou avoir) au présent + participe passé

Participes passés courants
  • 1er groupe (-ER) → -é : manger → mangé, jouer → joué
  • 2e groupe (-IR) → -i : finir → fini, choisir → choisi
  • Irréguliers : être → été, avoir → eu, faire → fait, aller → allé
Avoir ou être ?
  • Avoir : la plupart des verbes → J'ai mangé, tu as joué
  • Être : verbes de déplacement et pronominaux → Je suis allé(e), elle est venue
  • Verbes avec être : aller, venir, partir, arriver, entrer, sortir, naître, mourir, monter, descendre, rester, tomber…

📌 À retenir

  • 1er groupe (-ER) présent : -e, -es, -e, -ons, -ez, -ent
  • Imparfait : radical + -ais/-ais/-ait/-ions/-iez/-aient
  • Futur : infinitif + -ai/-as/-a/-ons/-ez/-ont
  • Passé composé : avoir/être + participe passé
