CE2

📝 Conjugaison

Présent, imparfait, futur, passé composé, impératif

📌 Le présent
Action actuelle ou habituelle. Verbes irréguliers à connaître : être, avoir, aller, venir, faire, voir, pouvoir, prendre, vouloir.
Ex : je vais · il vient · elle fait · nous avons · ils sont
📌 L'imparfait
Action passée qui durait ou se répétait. Terminaisons : -ais / -ais / -ait / -ions / -iez / -aient.
Ex : Il jouait chaque jour. · Nous chantions. · Elle était grande.
📌 Le futur simple
Action future. Terminaisons : -rai / -ras / -ra / -rons / -rez / -ront (ajoutées à l'infinitif).
Ex : Je chanterai. · Ils partiront. · Tu seras là.
📌 Le passé composé
Action terminée. Formule : auxiliaire avoir ou être (au présent) + participe passé.
Ex : J'ai mangé. · Elle est venue. · Nous avons joué.
📌 L'impératif
Ordre ou conseil. 3 personnes seulement (tu, nous, vous). Verbes en -er : pas de -s à "tu".
Ex : Mange ! · Partons ! · Finissez ! · Sois sage !

📌 À retenir

  • Présent : formes irrégulières (être/avoir/aller/faire…) à connaître
  • Imparfait : terminaisons en -ais / -ait / -ions / -aient
  • Futur : infinitif + -rai / -ras / -ra / -rons / -rez / -ront
  • Passé composé : avoir ou être + participe passé
