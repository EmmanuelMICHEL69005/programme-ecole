✍️ Orthographe

PP avec avoir, même, accords complexes

📌 PP avec avoir — accord si COD avant
Le participe passé conjugué avec avoir s'accorde avec le COD seulement si ce COD est placé AVANT le verbe. "Que" = toujours COD = toujours avant.
Ex : La lettre que j'ai écrite. · Les films que nous avons vus. · J'ai écrit une lettre. (pas d'accord)
📌 PP avec pronoms COD
Les pronoms le/la/les peuvent être COD placés avant le verbe → accord du PP. Identifier le genre et le nombre du pronom pour l'accord.
Ex : Je les ai vus. (les = masc. pl. → vus) · Je l'ai vue. (la = fém. → vue)
📌 "Même"
Adjectif (variable) : s'accorde, sens = identique ou renforce un pronom. Adverbe (invariable) : sens = aussi, jusqu'à, y compris.
Ex : Les mêmes erreurs. (adj.) · Eux-mêmes. (adj.) · Même les grands se trompent. (adv.)
📌 Accord adj. avec plusieurs noms
Si au moins un nom est masculin → accord au masculin pluriel. Si tous les noms sont féminins → accord au féminin pluriel.
Ex : une robe et un manteau bleus (masc. pl.) · ma mère et ma sœur sont heureuses (fém. pl.)

📌 À retenir

  • PP avec avoir → accord uniquement si COD avant le verbe
  • "que" = COD = toujours avant → toujours accord
  • "même" : adjectif (variable) / adverbe (invariable)
  • Adj. + noms mixtes → masculin pluriel
