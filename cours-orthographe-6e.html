PP avec avoir, même, homophones

1. Accord du PP avec avoir — le COD antéposé

Rappel de la règle

Avec l'auxiliaire avoir, le PP s'accorde avec le COD seulement si le COD est placé AVANT le verbe.

Le pronom relatif "que" = COD toujours placé avant

Quand une subordonnée relative est introduite par "que", ce "que" est le COD et est placé avant le verbe → le PP s'accorde avec l'antécédent de "que".

PhraseAntécédent de "que"Accord PP
La chanson que j'ai chantée."chanson" → fém. sing.chantée
Les livres que tu as lus."livres" → masc. plur.lus
Les fleurs que nous avons cueillies."fleurs" → fém. plur.cueillies
Avec les pronoms COD (le, la, les, l')

Cette robe, il l'a achetée. → "l'" = "la robe" (fém. sing.) → achetée
Ces garçons, je les ai vus. → "les" = garçons (masc. plur.) → vus

2. "Même" — adjectif ou adverbe ?

Adjectif → variable

Quand "même" signifie identique, similaire → il s'accorde.

  • Nous avons les mêmes goûts. (= goûts identiques)
  • Ces enfants eux-mêmes l'ont dit. (renforce le pronom)
Adverbe → invariable

Quand "même" signifie aussi, y compris → invariable.

  • Même les enfants le savent. (= y compris les enfants)
  • Il travaille même le dimanche.

4. Accord de l'adjectif avec plusieurs noms

Règle

Un adjectif qui qualifie plusieurs noms se met au pluriel.
Si les noms sont de genres différents → masculin pluriel l'emporte.

  • Un chapeau et une robe rouges. (accord au pluriel)
  • Des garçons et des filles contents. (masc. plur.)

📌 À retenir

  • PP + avoir : accord avec le COD si COD avant le verbe → "que" = toujours avant
  • "même" adj. = identique (variable) ; "même" adv. = aussi/y compris (invariable)
  • Adj. + 2 noms différents → pluriel ; si genres différents → masculin pluriel
  • quand = lorsque (temps) · quant = en ce qui concerne · qu'en = que + en
