🔤 Nom : désigne une personne, un animal, une chose (chat, maison, Lucas)
🔵 Déterminant : accompagne le nom (le, la, les, un, une, des, mon, ma…)
🟢 Verbe : exprime une action ou un état (manger, être, courir)
🟡 Adjectif : précise le nom (grand, rouge, joli)
🔴 Pronom : remplace un nom (je, tu, il, elle, nous, vous, ils, elles)
📏 GN : déterminant + nom (+ adjectif éventuellement) → le grand chat
