La phrase · Nom, verbe · Genre et nombre

1. La phrase

Majuscule au début → mots dans l'ordre → . ? ! à la fin
Règles
  • Une phrase commence par une lettre majuscule.
  • Une phrase se termine par un point . ? ou !
  • Une phrase a un sens complet.

✅ Le chat dort.    ❌ chat dort le

2. Le nom, le verbe, le déterminant

Le nom

Le nom désigne une personne, un animal, une chose, un lieu.
On peut mettre le / la / les / un / une devant un nom.

le chat · une maison · des enfants · l'école

Le verbe

Le verbe indique l'action ou l'état. Il change selon le sujet.

je mange · tu manges · il mange

Le déterminant

Le déterminant accompagne le nom. Il indique le genre et le nombre.

SingulierPluriel
Masc.le · unles · des
Fém.la · uneles · des

3. Masculin et féminin

Repérer le genre

Le déterminant indique si le nom est masculin ou féminin.

MasculinFéminin
le soleil · un livre · un chatla lune · une maison · une chatte
Former le féminin d'un nom

Souvent on ajoute un -e : un chat → une chatte · un ami → une amie
Parfois c'est différent : un cheval → une jument

4. Singulier et pluriel

Règle de base

Singulier = un seul → un chat · la maison
Pluriel = plusieurs → des chats · les maisons

Au pluriel on ajoute souvent un -s au nom (et à l'adjectif).
On change aussi le déterminant : le/la → les · un/une → des

📌 À retenir

  • Phrase : majuscule + sens complet + point final
  • Nom : précédé de le/la/un/une
  • Verbe : change avec le sujet (je mange · tu manges · il mange)
  • Masculin = le/un · Féminin = la/une
  • Pluriel = les/des (souvent + -s au nom)
