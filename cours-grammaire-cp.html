🔤 Grammaire

La phrase · Nom, verbe · Genre et nombre

📌 La phrase
Une phrase commence par une majuscule et se termine par un point (. ? !). Elle a un sens.
Ex : ✅ Le chat dort. · ❌ chat dort le
📌 Le nom
Le nom désigne une personne, un animal ou une chose. On peut mettre le / la / un / une devant.
Ex : le chat · une maison · un enfant
📌 Le verbe
Le verbe indique une action ou un état. Il change selon qui fait l'action.
Ex : je cours · tu cours · il court
📌 Masculin / Féminin · Singulier / Pluriel
Masculin : le / un · Féminin : la / une. Singulier = 1 · Pluriel = plusieurs (on ajoute souvent -s).
Ex : le soleil (masc.) · la lune (fém.) · un chat → des chats

📌 À retenir

  • Phrase : majuscule au début + point à la fin
  • Nom : le / la / un / une devant
  • Verbe : change avec le sujet
  • Masculin = le/un · Féminin = la/une
  • Pluriel = les/des (souvent + -s)
