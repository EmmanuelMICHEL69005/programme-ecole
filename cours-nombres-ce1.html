🔢 Les nombres jusqu'à 1000

Centaines, dizaines, unités — lire, écrire, comparer

1. Centaines, dizaines, unités

Définition

Un nombre est formé de centaines (C), de dizaines (D) et d'unités (U).
On les range dans un tableau de numération.

CentainesDizainesUnités
347
347 = 3 centaines + 4 dizaines + 7 unités
Exemples
  • 245 = 2 centaines + 4 dizaines + 5 unités = 200 + 40 + 5
  • 308 = 3 centaines + 0 dizaine + 8 unités = 300 + 0 + 8
  • 70 = 0 centaine + 7 dizaines + 0 unité = 70
Attention !

308 : le 0 des dizaines est important — il "garde la place" !

3. Comparer les nombres

Méthode

On compare les chiffres de gauche à droite : centaines d'abord, puis dizaines, puis unités.

Exemples avec < > =
  • 456 < 523 : 4 centaines < 5 centaines → 456 est plus petit
  • 738 > 729 : même centaine (7), même dizaine (3=2? Non, 3 > 2) → 738 est plus grand
  • 345 = 345 : tous les chiffres sont identiques
Astuce : le crocodile

Le signe < ou > "ouvre la bouche" vers le plus grand nombre — comme un crocodile affamé !

4. Ranger les nombres

Définitions

Ordre croissant = du plus petit au plus grand (↗)
Ordre décroissant = du plus grand au plus petit (↘)

Exemple

Ranger 342, 97, 500, 124, 287 dans l'ordre croissant :
→ 97 < 124 < 287 < 342 < 500

Droite des nombres de 0 à 1000

5. Encadrer un nombre

Définition

Encadrer un nombre, c'est trouver les deux dizaines (ou centaines) les plus proches entre lesquelles il se trouve.

Exemples
  • Encadrer 347 à la dizaine : 340 < 347 < 350
  • Encadrer 347 à la centaine : 300 < 347 < 400
  • Intercaler un nombre : entre 250 et 260 → 255 par exemple

6. Nombres pairs et impairs

Définition

Un nombre pair se termine par 0, 2, 4, 6 ou 8.
Un nombre impair se termine par 1, 3, 5, 7 ou 9.

Pairs
0 2 4 6 8
terminaisons
Impairs
1 3 5 7 9
terminaisons
Pourquoi c'est utile ?

Un nombre pair peut être partagé en deux groupes égaux. Un nombre impair, non — il reste 1 en dehors.

Exemples rapides
  • 348 : se termine par 8 → pair
  • 507 : se termine par 7 → impair
  • 1000 : se termine par 0 → pair

7. Les doubles et moitiés

Définition

Le double d'un nombre = ce nombre + ce nombre (× 2).
La moitié = partager en 2 groupes égaux (÷ 2).

Moitié de

Moitié de

Attention !

On ne peut calculer la moitié que des nombres pairs. La moitié de 7 n'est pas un nombre entier.

📌 À retenir

  • Centaines — Dizaines — Unités : lire de gauche à droite
  • Pour comparer : regarder les centaines d'abord, puis les dizaines, puis les unités
  • Pair : se termine par 0, 2, 4, 6 ou 8
  • Impair : se termine par 1, 3, 5, 7 ou 9
  • Double × 2, moitié ÷ 2
