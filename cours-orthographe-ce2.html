Homophones, pluriels, accords

1. Homophones a / as / à

MotNatureAstuceExemple
averbe avoir (il/elle/on)Remplace par avait → ça marcheIl a faim. → Il avait faim. ✓
asverbe avoir (tu)Remplace par avais → ça marcheTu as froid. → Tu avais froid. ✓
àprépositionRemplace par avait/avais → ça ne marche pasJe vais à Paris. → ✗ vais avait Paris

MotNatureAstuceExemple
estverbe être (il/elle/on)Remplace par était → ça marcheIl est beau. → Il était beau. ✓

4. Pluriel des noms

Règle générale

La plupart des noms prennent un -s au pluriel : chat → chats

Cas particuliers
  • Noms en -al → -aux : cheval → chevaux, journal → journaux
  • Noms en -eau → -eaux : bateau → bateaux, gâteau → gâteaux
  • Noms en -ou (7 exceptions) → -oux : hibou → hiboux, genou → genoux
  • Noms en -eu → -eux : jeu → jeux, feu → feux
  • Noms déjà en -s, -z, -x → invariables : bras, nez, voix
Les 7 noms en -ou qui font -oux

Bijou, caillou, chou, genou, hibou, joujou, pou → bijoux, cailloux, choux…

5. Accord de l'adjectif

Règle

L'adjectif s'accorde en genre et en nombre avec le nom qu'il qualifie.

Formation du féminin des adjectifs
  • Règle générale : + -e → grand → grande
  • Déjà en -e : rouge → rouge (invariable)
  • -eux → -euse : heureux → heureuse
  • -er → -ère : premier → première
  • -el / -en → double consonne + e : cruel → cruelle, ancien → ancienne
  • -ien → -ienne, -on → -onne, -et → -ette
  • Irréguliers : beau → belle, blanc → blanche, doux → douce

6. -é ou -er ? (infinitif ou participe passé)

Règle

Après un auxiliaire conjugué (avoir ou être), on écrit le participe passé en -é.
Dans les autres cas, c'est souvent l'infinitif en -er.

Astuce : le test avec "vendre/vendu"

Remplace le verbe par vendre ou vendu :
→ si on peut dire vendu → c'est le participe passé → on écrit -é
→ si on peut dire vendre → c'est l'infinitif → on écrit -er

PhraseTestForme correcte
Il a mangé une pomme.Il a vendu → ✓ venduparticipe passé → -é
Il va manger une pomme.Il va vendre → ✓ vendreinfinitif → -er
Elle vient de chanter.vient de vendre → ✓ vendreinfinitif → -er
Nous avons joué au foot.avons vendu → ✓ venduparticipe passé → -é
Attention aux pièges
  • Il faut chanter (= vendre) → infinitif
  • Il a bien chanté (= vendu) → participe passé
  • Pour jouer au foot (= pour vendre) → infinitif

📌 À retenir

  • a / à : remplace par "avait" → si ça marche → "a" sans accent
  • est / et : remplace par "était" → si ça marche → "est"
  • son/sont et on/ont : remplace par "étaient/avaient" pour vérifier
  • Pluriel : généralement -s, mais -al→-aux, -eau→-eaux, 7 -ou→-oux
  • Adjectif : s'accorde en genre et nombre avec le nom
  • -é ou -er : remplace par "vendu/vendre" → si vendu ✓ → -é ; si vendre ✓ → -er
