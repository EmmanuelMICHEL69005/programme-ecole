✏️ Orthographe

Homophones grammaticaux · accord de l’adjectif

⚡ son / sont
son = déterminant possessif. sont = verbe être (remplacer par étaient).
Son chat dort. → mon chat ✅ → possessif
Ils sont partis. → ils étaient partis ✅ → verbe être
💡 Test sont : remplacer par étaient. Test son : remplacer par mon.
⚡ on / ont
on = pronom (remplacer par il). ont = verbe avoir (remplacer par avaient).
On mange bien ici. → Il mange bien ici ✅ → pronom
Ils ont fini. → ils avaient fini ✅ → verbe avoir
💡 Test ont : remplacer par avaient.
⚡ ou / où — ses / ces
ou / où
ou = choix (= ou bien)
Tu viens ou tu restes ?
où = lieu
Où habites-tu ?
ses / ces
ses = ses livres (possessif = mes)
ces = ces livres (démonstratif = ces…-là)
⚡ c’est / s’est
c’est = cela est. s’est = il/elle s’est (passé composé pronominal).
C’est beau. → Cela est beau ✅
Il s’est levé. → il s’était levé ✅ → passé composé
💡 Test s’est : remplacer par s’était. Test c’est : dire cela est.

📌 À retenir

  • son (possessif = mon) / sont (être → étaient)
  • on (pronom = il) / ont (avoir → avaient)
  • ou (choix = ou bien) / où (lieu)
  • ses (possessif = mes) / ces (démonstratif = ces…-là)
