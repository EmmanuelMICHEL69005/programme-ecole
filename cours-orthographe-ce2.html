✏️ Orthographe

Homophones, pluriels, accords

1. Homophones a / à

MotNatureAstuceExemple
averbe avoir (présent)Remplace par avait → ça marcheIl a faim. → Il avait faim. ✓
àprépositionRemplace par avait → ça ne marche pasJe vais à Paris. → ✗ vais avait Paris

2. Homophones est / et

MotNatureAstuceExemple
estverbe être (présent)Remplace par était → ça marcheIl est beau. → Il était beau. ✓
etconjonction de coordinationRemplace par et puis → ça marcheIl mange et boit. → ... et puis boit. ✓

3. Homophones son / sont et on / ont

MotNatureAstuceExemple
sondéterminant possessifRemplace par mon/tonIl cherche son sac. → mon sac ✓
sontverbe être (présent)Remplace par étaientIls sont partis. → étaient ✓
onpronom sujetRemplace par ilOn mange. → Il mange. ✓
ontverbe avoir (présent)Remplace par avaientIls ont fini. → avaient ✓

4. Pluriel des noms

Règle générale

La plupart des noms prennent un -s au pluriel : chat → chats

Cas particuliers
  • Noms en -al → -aux : cheval → chevaux, journal → journaux
  • Noms en -eau → -eaux : bateau → bateaux, gâteau → gâteaux
  • Noms en -ou (7 exceptions) → -oux : hibou → hiboux, genou → genoux
  • Noms en -eu → -eux : jeu → jeux, feu → feux
  • Noms déjà en -s, -z, -x → invariables : bras, nez, voix
Les 7 noms en -ou qui font -oux

Bijou, caillou, chou, genou, hibou, joujou, pou → bijoux, cailloux, choux…

5. Accord de l'adjectif

Règle

L'adjectif s'accorde en genre et en nombre avec le nom qu'il qualifie.

Formation du féminin des adjectifs
  • Règle générale : + -e → grand → grande
  • Déjà en -e : rouge → rouge (invariable)
  • -eux → -euse : heureux → heureuse
  • -er → -ère : premier → première
  • -el / -en → double consonne + e : cruel → cruelle, ancien → ancienne
  • -ien → -ienne, -on → -onne, -et → -ette
  • Irréguliers : beau → belle, blanc → blanche, doux → douce

📌 À retenir

  • a / à : remplace par "avait" → si ça marche → "a" sans accent
  • est / et : remplace par "était" → si ça marche → "est"
  • son/sont et on/ont : remplace par "étaient/avaient" pour vérifier
  • Pluriel : généralement -s, mais -al→-aux, -eau→-eaux, 7 -ou→-oux
  • Adjectif : s'accorde en genre et nombre avec le nom
