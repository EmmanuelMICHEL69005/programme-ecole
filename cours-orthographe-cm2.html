Accord PP avec avoir · homophones avancés

✅ Accord PP avec AVOIR : la règle du COD
Avec AVOIR, le PP s’accorde avec le COD s’il est placé avant le verbe.
J’ai mangé des pommes. [COD après] → pas d’accord
Les pommes que j’ai mangées. [COD = que = pommes fém. plur., avant] → accord !
Je les ai vues. [COD = les = fém. plur., avant] → accord !
💡 Méthode : trouver le COD → est-il AVANT le verbe ? Oui → accord. Non → pas d’accord.
⚡ quand / qu’en
quand = conjonction de temps (= lorsque). qu’en = que + en.
Quand il arrive, tout le monde se lève. → remplaçable par lorsque
Il ne le fera qu’en sachant la vérité. → que + en
💡 Test : si on peut remplacer par lorsque → écris quand.
⚡ quel(le) / qu’elle · près / prêt
quel(le)(s)
quel adj. interrogatif/exclamatif
Quel beau temps ! · Quelle chance !
qu’elle = que + elle
près / prêt
près = proche (≠ loin)
J’habite près de l’école.
prêt adj. = préparé
Je suis prêt(e) à partir.
📝 Féminins irréguliers des adjectifs
Terminaison masc.Terminaison fém.Exemples
-(e)r-èredernier → dernière · léger → légère
-(e)u-euseheureux → heureuse · vieux → vieille
-(i)f-iveactif → active · naïve
-(i)en-ienneancien → ancienne · musicien → musicienne

📌 À retenir

  • PP avec AVOIR : accord si COD avant le verbe
  • quand (= lorsque) / qu’en (que + en)
  • quel adj. accordé / qu’elle pronom
  • près (lieu) / prêt adj. (préparé)
