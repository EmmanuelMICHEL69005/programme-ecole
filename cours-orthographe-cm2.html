PP avec avoir, homophones, chaîne d'accords

1. Accord du participe passé avec avoir

Règle

Avec l'auxiliaire avoir, le participe passé ne s'accorde pas avec le sujet.
Il s'accorde avec le COD seulement si celui-ci est placé AVANT le verbe.

Cas 1 : COD après le verbe → pas d'accord

Elle a mangé des pommes. → "pommes" est après → PP invariable : mangé

Cas 2 : COD avant le verbe (pronom, que…) → accord

Les pommes qu'elle a mangées. → "que" = les pommes (fém. plur.) est avant → mangées
Il les a reçues. → "les" = les pommes (fém. plur.) avant → reçues

Règle "de" devant adj. pluriel

Devant un adjectif qui commence par une consonne, au pluriel : de et non des
de beaux chevaux · de vieux amis · de nombreux problèmes

📌 À retenir

  • PP + avoir : accord avec le COD si COD est AVANT le verbe
  • La lettre que → accord avec le nom antécédent de "que"
  • quand = lorsque · quant = en ce qui concerne · qu'en = que + en
  • davantage = plus (adverbe) · d'avantage(s) = de + nom
  • leur invariable quand pronom · leurs avec -s quand déterminant pluriel
