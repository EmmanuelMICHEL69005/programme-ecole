Accords, participe passé, homophones

1. La chaîne d'accords dans le GN

Principe

Dans un groupe nominal, le nom noyau impose son genre et son nombre au déterminant et aux adjectifs.

Exemple : les beaux vieux chevaux
→ "chevaux" : masculin pluriel → dét. "les", adj. "beaux", adj. "vieux" sont au masc. plur.

2. L'accord sujet-verbe

Règle de base

Le verbe s'accorde toujours avec son sujet — en genre non, mais en nombre et personne.

  • Sujet inversé : Chantent dans la nuit les oiseaux. → sujet = oiseaux → chantent ✓
  • Sujet éloigné : Les élèves de ma classe adorent la musique. → sujet = élèves → adorent ✓
  • Deux sujets : Paul et Marie partent. → sujet = Paul + Marie → 3e pers. pluriel ✓

3. Le participe passé avec l'auxiliaire être

Règle

Quand le verbe est conjugué avec être, le participe passé s'accorde avec le sujet.

SujetExemple
masc. sing.Paul est parti.
fém. sing.Léa est partie.
masc. plur.Les garçons sont partis.
fém. plur.Les filles sont parties.
Avec avoir

Avec l'auxiliaire avoir, le participe passé ne s'accorde pas avec le sujet :
Elle a chanté. / Ils ont mangé. / Nous avons pris.

5. ces / ses

Distinguer
  • ces = déterminant démonstratif → Je veux ces bonbons. (→ remplacer par "les")
  • ses = déterminant possessif → Il prend ses affaires. (→ remplacer par "mes")

📌 À retenir

  • GN : dét. + adj. → tout s'accorde avec le nom noyau
  • Pluriels en -aux : cheval → chevaux, journal → journaux
  • PP + être → accord avec le sujet (genre + nombre)
  • a/à : remplacer par "avait" → si ça marche = "a" ; sinon = "à"
  • on/ont : remplacer par "il" → si ça marche = "on" ; sinon = "ont"
  • son/sont : remplacer par "mon" → si ça marche = "son" ; sinon = "sont"
