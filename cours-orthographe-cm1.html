✏️ Orthographe

Homophones complexes · accord PP avec être

⚡ leur / leurs
leur invariable = pronom COI (= lui). leurs = adjectif possessif pluriel (accord avec le nom).
Je leur parle. → Je lui parle ✅ → pronom, invariable
Ils rangent leurs affaires. → leurs affaires (plusieurs) → adjectif, accord
💡 Test : si on peut remplacer par lui → pronom → leur sans s.
⚡ tout / toute / tous / toutes
FormeUsageExemple
toutadv. invariable (= entièrement)Elle est tout étonnée.
toutefém. sing.Toute la classe est là.
tousmasc. plur.Tous les enfants jouent.
toutesfém. plur.Toutes les filles chantent.
💡 Adverbe tout = entièrement. Invariable SAUF devant adj. fém. commençant par consonne : toute petite.
✅ Accord PP avec Être
Avec Être, le PP s’accorde en genre et nombre avec le sujet.
SujetPPExemple
Masc. sg.(rien)Il est parti.
Fém. sg.+eElle est partie.
Masc. plur.+sIls sont partis.
Fém. plur.+esElles sont parties.
💡 Les 14 verbes avec Être : aller, venir, partir, arriver, naître, mourir, rester, tomber… + tous les pronominaux.
📝 Préfixes et suffixes utiles
AffixeSensExemples
dé-/dés-action inversedéfaire, désordre
re-/ré-répétitionrefaire, relire, réutiliser
in-/im-/il-contraireimpossible, illisible, inconnu
anti-contreantidote, antibrouillard

📌 À retenir

  • leur (pronom = lui) invariable / leurs (adj. possessif) accordé
  • tout adv. invariable · toute/tous/toutes adj. accordés
  • PP avec Être : s’accorde avec le sujet
  • Préfixes : dé-/re-/in- changent le sens du mot
