🌍 Histoire-Géographie

Civilisations antiques, Grèce, Rome

📌 La Mésopotamie
Entre le Tigre et l'Euphrate (actuel Irak). Vers −3500 : naissance de l'écriture cunéiforme. Code d'Hammurabi (−1750) : premier code de lois écrit.
Ex : villes d'Ur, Babylone · ziggourats (temples-montagnes)
📌 L'Égypte antique
Le pharaon dirige et est considéré comme un dieu. Le Nil fertilise les terres. Écriture : hiéroglyphes. Monuments : pyramides et temples.
Ex : Toutankhamon · pyramide de Gizeh (−2560) · Cléopâtre
📌 La Grèce antique
Cités-états indépendantes. Athènes invente la démocratie (vote des citoyens). Mythologie et philosophie. Jeux Olympiques depuis −776.
Ex : Socrate · Platon · Périclès · Sparte vs Athènes
📌 Rome antique
République (−509) puis Empire (−27). Jules César étend le territoire. Auguste, 1er emperor. Droit romain, langue latine.
Ex : Colisée · Pont du Gard · Jules César assassiné en −44

📌 À retenir

  • Mésopotamie → écriture cunéiforme (−3500) · Code d'Hammurabi
  • Égypte → pharaon · Nil · pyramides · hiéroglyphes
  • Grèce → démocratie · cités-états · JO (−776)
  • Rome → République puis Empire · Jules César · Auguste
