🔢 Nombres entiers relatifs

Positifs, négatifs, valeur absolue, calculs

📌 Entiers relatifs
Positifs, négatifs, zéro. Ordre sur la droite numérique : les grands sont à droite. Valeur absolue |a| = distance à 0, toujours positive.
Ex : −5 < −2 < 0 < 3 < 8 · |−5| = 5 · température −3°C < 0°C < 20°C
📌 Addition et soustraction
Même signe : additionner les valeurs absolues, garder le signe. Signes opposés : soustraire, prendre le signe du plus grand. Soustraire b = additionner l'opposé de b.
Ex : −5 + 3 = −2 · −4 + (−3) = −7 · 3 − (−5) = 3 + 5 = 8
📌 Règles des signes × et ÷
Même signe → résultat positif. Signes différents → résultat négatif. Même règle pour la division.
Ex : −3 × 4 = −12 · −6 ÷ (−2) = +3 · (−3) × (−4) = +12

📌 À retenir

  • Entiers relatifs : positifs, négatifs, zéro
  • Valeur absolue = distance à 0 (toujours positive)
  • Même signe → additionner · signes différents → soustraire
  • Règles × et ÷ : même signe = + · signes différents = −
