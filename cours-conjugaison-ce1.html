Présent, futur proche, passé composé

1. L'infinitif

Définition

L'infinitif est la forme de base du verbe — celle qu'on trouve dans le dictionnaire. Le verbe n'est pas conjugué.

Les terminaisons de l'infinitif
  • -ER (1er groupe) : chanter, manger, jouer, sauter (la plupart des verbes)
  • -IR (2e groupe) : finir, grandir, choisir (participe présent en -issant)
  • Autres (3e groupe) : faire, être, avoir, aller, venir, prendre…

3. Présent des verbes du 1er groupe (-ER)

Terminaisons

Exemples avec d'autres verbes -ER
  • jouer : je joue, tu joues, il joue, nous jouons, vous jouez, ils jouent
  • manger : je mange, tu manges, nous mangeons (+ e avant -ons)

4. Le futur proche

Formation

Verbe aller (au présent) + infinitif du verbe

Exprime une action qui va se passer très bientôt.

PronomFutur proche (chanter)
jeje vais chanter
tutu vas chanter
il / elleil/elle va chanter
nousnous allons chanter
vousvous allez chanter
ils / ellesils/elles vont chanter
Exemples
  • Je vais manger. → dans quelques instants
  • Il va pleuvoir. → bientôt
  • Nous allons partir. → très prochainement

5. Passé composé

Formation (avec avoir)

Avoir (au présent) + participe passé

Pour les verbes du 1er groupe (-ER), le participe passé se forme avec -é à la place de -er.

Participes passés -ER → -é
  • chanter → chanté
  • jouer → joué
  • manger → mangé
  • regarder → regardé
PronomPassé composé (manger)
j'j'ai mangé
tutu as mangé
il / elleil/elle a mangé
nousnous avons mangé
vousvous avez mangé
ils / ellesils/elles ont mangé

📌 À retenir

  • 1er groupe (-ER) au présent : je chante, tu chantes, il chante, nous chantons, vous chantez, ils chantent
  • être : suis / es / est / sommes / êtes / sont
  • avoir : ai / as / a / avons / avez / ont
  • Futur proche : aller (présent) + infinitif → je vais chanter
  • Passé composé : avoir + participe passé en -é → j'ai chanté
